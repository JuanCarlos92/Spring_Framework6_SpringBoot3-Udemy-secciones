# Spring Framework 6 & Spring Boot 3: De Cero a Experto

Este repositorio contiene el código fuente, ejercicios y proyectos desarrollados a lo largo del curso "Spring Framework 6 & Spring Boot 3 desde cero a experto". El contenido está organizado estrictamente por secciones y subdirectorios para reflejar el progreso del aprendizaje de forma estructurada.

## Resumen del Repositorio
* Total Secciones: 21
* Clases Completadas: 273
* Duración Total: ~40.5 Horas
* Enfoque: Java Moderno (17+), Spring Boot 3, Seguridad JWT/OAuth2 y Desarrollo Full Stack.

---

## Estructura de Directorios

El repositorio sigue la siguiente jerarquía de carpetas, donde cada sección contiene sus respectivos apartados y código:

### Bloque 1: Fundamentos y Lógica de Spring
* Sección 1: Introducción al curso (Configuración JDK, IDE y bases de Spring Boot).
* Sección 2: Spring MVC (Arquitectura y manejo de Vistas/Controladores).
* Sección 3: Inyección de Dependencias (IoC).
* Sección 4: Inyección de Dependencias: Ejemplo Factura.
* Sección 5: Manejo de excepciones.
* Sección 6: Interceptores HTTP.
* Sección 7: Interceptores HTTP Horario de acceso a clientes.
* Sección 8: Programación Orientada a Aspectos (AOP).

### Bloque 2: Persistencia y APIs RESTful
* Sección 9: JPA e Hibernate (CRUD y mapeo de bases de datos).
* Sección 10: JPA e Hibernate Eventos Ciclo de vida.
* Sección 11: JPA e Hibernate - Asociaciones (Relaciones OneToMany, ManyToOne, ManyToMany).
* Sección 12: Spring Web API RESTful - CRUD.

### Bloque 3: Seguridad y Despliegue (DevOps)
* Sección 13: Backend Users Spring Security JWT (Autenticación basada en Tokens).
* Sección 14: Spring Authorization Server OAuth 2.1.
* Sección 15: Deploy Amazon Web Services (AWS) EC2.
* Sección 16: Deploy en Apache Tomcat 10 Externo.

### Bloque 4: Full Stack Integration (React & Angular)
* Sección 17: React: Frontend App (Desarrollo del cliente con React).
* Sección 18: Backend App (Configuración del servidor para consumo Frontend).
* Sección 19: Comunicación entre Frontend React y Backend Spring.
* Sección 20: Angular: Frontend App (Desarrollo del cliente con Angular).
* Sección 21: Comunicación entre Frontend Angular y Backend Spring.
